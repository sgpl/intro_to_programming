What Is The Internet?

- The internet is a collection of HTML documents.
- It was invented in the early 1990s.
- It has about 30 billion webpages.

How The Internet Works:

The internet works by facilitating communication between different computers that are linked together. Essentially when a person uses a browser to request stuff on the internet, here is what happens:

- The browser sends a HTTP[1] request to a server via the internet.
- The server[2] responds to the request and it is relayed back to the browser.
- The browser interprets the information sent back and displays it to the user.

Definitions:

[1] HTTP: stands for Hypertext Transfer Protocol. It is a simple protocol that the browser uses for communicating with the servers.

[2] Servers: are computers that are optimized for hosting files / storing information as opposed to being a computer for personal use.

HTML

HTML stands for HyperText Markup Language. It is the building block of the web. It contains text (stuff you see), markup (that tells the browser how to display stuff), references to other documents, images, videos, etc. and links to other webpages.

HTML Tags And Elements

Tags in a HTML document are what we use to tell the browser how to display the contents that are affected by that particular tag. Most tags have two parts: an opening tag and a closing tag. Tags that don't have any contents and hence don't need a closing tag are called void tags.

Elements are what make up a HTML document. Tags and and the contents in between them (opening and closing tag), together, make up an element. The browser can identify the type of an element by looking at the tags.

HTML Attributes: some elements have additional values / information that provide the browser additional context. Attributes are contained in the opening tags of the elements. An example of one such attribute is the image tag that takes an attribute called alt that can give visually impaired users a description of the image.

Inline vs Block Elements: inline elements end a line whereas block elements make an invisible box around the text/contents and can have user defined height and width.

Container Elements do nothing but contain stuff inside themselves. They aid in creating web pages with complex structures.

DOM - Document Object Model - a cross-platform and language-independent convention for representing and interacting with objects in HTML (and other markup languages). The nodes of every document are organized in a tree structure, called the DOM tree.

A tree structure or tree diagram is a way of representing the hierarchical nature of a structure in a graphical form. It is named a "tree structure" because the classic representation resembles a tree, even though the chart is generally upside down compared to an actual tree, with the "root" at the top and the "leaves" at the bottom.
